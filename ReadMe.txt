Weather Prediction Project
Source of data: https://www.weather.gov 
To run:
1. Install Python 3.10+
2. Install libraries: pip install numpy tensorflow keras matplotlib pyinstaller
3. Run: python weather_predict.py
4. To build EXE: pyinstaller -F weather_predict.py (EXE will be in dist folder, move it to build folder)
Prediction saved in prediction.txt
